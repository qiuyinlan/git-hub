# 学生成绩
```
本题要求编写程序，计算学生们的平均成绩，并统计及格（成绩不低于60分）的人数。题目保证输入与输出均在整型范围内。

输入格式:
输入在第一行中给出非负整数N，即学生人数。第二行给出N个非负整数，即这N位学生的成绩，其间以空格分隔。 

输出格式:
按照以下格式输出：

average = 成绩均值
count = 及格人数
其中平均值精确到小数点后一位。 
```
